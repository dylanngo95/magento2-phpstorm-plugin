events.xml file is the place where module's Observers are declared to adjust the system.

Read more about Subscribing to events.

Areas where events can be used:
Area Area Code Description
Global global Observer will be executed in all areas: adminhtml, crontab, frontend, graphql, webapi_rest, webapi_soap.
Admin area adminhtml Observer will be executed in the adminhtml area only.
Storefront frontend Observer will be executed in the frontend area only.
Cron crontab Observer will be executed in the crontab area only.
GraphQl graphql Observer will be executed in the graphql area only.
REST API webapi_rest Observer will be executed in the webapi_rest area only.
SOAP API webapi_soap Observer will be executed in the webapi_soap area only,
