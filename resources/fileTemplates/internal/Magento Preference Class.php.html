The object manager uses the preference mappings to determine what the default implementation is for that class for a particular scope. For example, a preference is used to override a public or protected method from a core class.
Read more About the preferences in the DevDocs.
Predefined variables explanation:
${NAMESPACE}   Preference's namespace.
${USE}   The namespace of the PHP class to extend from.
${NAME}   The name of PHP class.
${EXTENDS}   The name of the PHP class to extend from.
